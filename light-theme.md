## Color Palette

### Background Colors
```css
--bg-primary:    /* Main page background */
--bg-secondary:  /* Card/section backgrounds */
--bg-elevated:   /* Hover states, elevated elements */
```

### Text Colors
```css
--text-primary:   /* Main text */
--text-secondary: /* Secondary/muted text */
--text-muted:     /* Subtle text, placeholders */
```

### Accent Colors
```css
--accent:       /* Primary brand color */
--accent-dark:  /* Darker variant for hover */
--accent-light: /* Lighter variant */
```

### Status Colors
```css
--success: /* Success messages, confirmations */
--warning: /* Warnings */
--error:   /* Errors, destructive actions */
--info:    /* Informational messages */
```

### Border Colors
```css
--border-color: /* Default borders */
--border-hover: /* Borders on hover */
```
